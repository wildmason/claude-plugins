# Wildmason Claude Plugins

Claude Code plugins from [Wildmason](https://wildmason.dev).

## Add this marketplace

```
/plugin marketplace add wildmason/claude-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [swarm](plugins/swarm/) | Multi-agent swarm orchestration — review, implement, debug, and research modes |

## Install a plugin

```
/plugin install swarm@wildmason/claude-plugins
```
